Task Management Web Application

Project Overview

This is a Task Management Web Application developed using HTML, CSS, and JavaScript.

The application allows users to:

- Create tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Store tasks using Local Storage

Features

Task Creation

Users can add new tasks with:

- Title
- Description
- Priority Level

Task Tracking

Users can:

- View all tasks
- Mark tasks as completed
- Undo completed tasks

Task Deletion

Users can permanently remove tasks.

Data Persistence

Tasks are stored in browser Local Storage and remain available after refreshing the page.

Responsive Design

The application works on desktop and mobile devices.

Technologies Used

- HTML5
